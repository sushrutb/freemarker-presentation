Wisebuy.in

What is Wisebuy.in?

  • Shopping search engine
  • 1 Million+ products
  • From 50+ stores
  • 80+ categories

Why Freemarker?

  • Simple
  • Powerful
    • Code reuse - Macros
    • Template libraries - for larger projects
    • Arbitrary data models
  • Fast
    • Caching
    • Fast page assembly
  • Testing of markup is easy

Template Structure

	
		<html>
		<head>
		  <title>Welcome!</title>
		</head>
		<body>
		  <#-- Greet the user with his/her name -->
		  <h1>Welcome ${user}!</h1>
		  <p>We have these animals:
		  <ul>
		  <#list animals as being>
		    <li>${being.name} for ${being.price} Euros
		  </#list>
		  </ul>
		</body>
		</html>

Directives

  • if / else /elseif
  • switch / case / default / break
  • list / break
  • include / import
  • function / return
  • macro / nested / return
  • visit / recurse / fallback - used for xml processing

Macro and nesting

	
			<#macro repeat count>
			  <#list 1..count as x>
			    <#nested x, x/2, x==count>
			  </#list>
			</#macro>
			<@repeat count=4 ; c, halfc, last>
			  ${c}. ${halfc}<#if last> Last!</#if>
			</@repeat>

Templating templates

	
		<#macro page_html> 
		<!DOCTYPE html>
		<html lang="en">
			<@page_head/>
		    <body>
		    	<@navbar/>
		    	<@productmenu/>
				<form name="mainForm" class="form-inline" method="GET" action="/products">
				<div style="margin-top:20px;margin-bottom:20px;">    	
			   		<@searchbar/>
		   		</div>
		   		<@page_view/>
		    	</form>
				<@aboutcontent/>
		   		<@footer/>
		   		<@social_buttons/>
		    </body>
		</html>
		</#macro>

  • Creating re-usable components
